ソースコードの場所と内容


--------------------------------------------------------------------------------------------------------

「game」 フォルダには私が制作したプログラムのソースコードがあります。

--------------------------------------------------------------------------------------------------------

「game」 フォルダにある 「src」 の中身全てが私が制作した箇所です。

--------------------------------------------------------------------------------------------------------

「game」 フォルダ内の 「data」 フォルダにはゲームに必要なスプライト（画像）やサウンド、外部データファイルなどがあります。

--------------------------------------------------------------------------------------------------------

「csv」
・enemy_table.csv		// 敵の種類、出現場所、出現時間がデータ化してあります。
・sound_file.csv		// 読み込んでおくサウンドファイル名がデータ化してあります。

--------------------------------------------------------------------------------------------------------

「sound」
音楽データは著作権フリーまたは、無償で利用可能な素材を使用しています

--------------------------------------------------------------------------------------------------------

「bgm」（クレジット必須のため記載）
・Oblivion.mp3		// TheFatRat - Oblivion (feat. Lola Blanc)							
				// LINK : https://youtu.be/Gc3tqnhmf5U

--------------------------------------------------------------------------------------------------------

「texture」
全て「paint.net」を使用して自作しました。

--------------------------------------------------------------------------------------------------------

「src」
main.cpp 			// ※ メインループがあります ※

--------------------------------------------------------------------------------------------------------

「Game」			// ゲームに関わるフォルダ

「Game」
・「GameObject」 		// ゲーム内で使用するオブジェクト

--------------------------------------------------------------------------------------------------------

「GameObject」
・「BulletManager」		// 弾を敵味方問わず管理するプログラム
・「CameraManager」		// 画面振動などで使用するプログラム
・「EffectManager」		// エフェクトの生成や消滅などを管理するプログラム
・「Field」			// 背景スクロール処理を行うプログラム
・「InterfaceManager」	// 下部のメッセージやスコアのUIを生成や消滅などを管理するプログラム
・「ItemManager」		// アイテムを生成や消滅などを管理するプログラム
・「SceneManager」		// シーン遷移を管理するプログラム
・「SoundManager」		// 効果音やBGMの再生、停止、音量調整などを管理するプログラム
・「UnitManager」		// 敵の生成や当たり判定などを管理するプログラム
